OpenClaw Setup in 30 Minutes

Get your own AI agent running on your Mac. No terminal experience needed. No monthly fees. Just follow the steps.

⏱ 30 MIN READ 💻 MACOS 🆓 FREE Updated April 2026
01

What You're Building

OpenClaw is an AI agent that lives on your machine 24/7. You talk to it through Telegram, WhatsApp, or Discord — and it can browse the web, manage files, send emails, run code, and remember everything across sessions.

Think of it like having a smart assistant who never sleeps, has access to your computer, and proactively checks in on things.

Why this matters: Your data stays local. No third-party server stores your conversations. You own everything.
02

What You Need

  • Mac (macOS 13 or later)
  • Telegram account (free — easiest setup)
  • AI API key — at least one:
    • OpenAI (GPT) — $5 credit on signup
    • Anthropic (Claude) — free tier available
    • Google (Gemini) — free tier available
Cost: The AI model costs pennies per interaction. Expect $2-10/month for casual use. You can also use free models via Ollama (local, zero cost).
03

Install OpenClaw

Open your Terminal (Cmd + Space → type "Terminal") and run:

# Install via npm (requires Node.js)
npm install -g openclaw

# If you don't have Node.js:
brew install node

# Then install OpenClaw
npm install -g openclaw

That's it. One command and you're in.

04

Connect Your AI Model

Set your API key so OpenClaw knows which AI brain to use:

# Set OpenAI key
openclaw config set models.openai.key sk-your-key-here

# Or Anthropic (Claude)
openclaw config set models.anthropic.key sk-ant-your-key-here

# Or Google (Gemini)
openclaw config set models.google.key your-key-here

You can use multiple providers and switch between them. OpenClaw routes to the best model for each task.

05

Connect Telegram

The fastest way to interact with your agent. Setup takes 2 minutes:

  • Open Telegram → search @BotFather
  • Send /newbot
  • Name your bot → get the API token
  • Run this in your terminal:
openclaw config set channels.telegram.token your-bot-token-from-botfather
openclaw gateway start

Your bot is now live. Message it in Telegram and it responds.

06

Customize Your Agent

OpenClaw creates a workspace folder. This is where your agent's personality, memory, and tools live:

# Open your workspace
open ~/.openclaw/workspace

# Key files:
# SOUL.md → Your agent's personality
# MEMORY.md → Long-term memory
# USER.md → Info about you
# TOOLS.md → Environment-specific notes

Edit SOUL.md to define how your agent talks, what it values, and how it behaves. This is where the magic happens — your agent becomes yours.

07

Test It Out

Try these messages in Telegram to see what your agent can do:

  • "What's the weather in Philadelphia?" — real-time data
  • "Search the web for the latest AI news" — web browsing
  • "Set a reminder for 2pm tomorrow" — scheduled tasks
  • "Read my desktop and organize the files" — file management
  • "Create a landing page for my business" — code generation
Pro tip: Your agent gets smarter over time. It remembers conversations, learns your preferences, and builds context. The more you use it, the more powerful it becomes.
08

Keep It Running 24/7

By default, the gateway stops when you close Terminal. To keep it alive:

# Start as a background service
openclaw gateway start --daemon

# Check status
openclaw gateway status

# View logs
openclaw gateway logs

For true 24/7 uptime, run OpenClaw on a VPS (DigitalOcean, Linode — $5/mo) or a Raspberry Pi. Same setup, just on a server that never sleeps.
